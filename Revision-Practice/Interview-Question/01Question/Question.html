Q.1 What is JavaSCript? How to use it?

  • Ans. -> JavaScript ek programming or Scripting languages hai. Jo ek Webpage me functionality ko add karta hai(Example => Clickable buttons,Popup menus,complex animation etc.)
  • JavaScript "Single Threaded language" hai jo ek baar me sirf ek kam hi karti hai, apne pahle kisi kam ko pura karne ke baad hi yeh apne alge kam ko karti hai.
  • JavaScript se ham HTML or CSS dono ko update or change kar sakte hai.
  • Abhi ham JavaScript ke "ES6" version ka use karte hai.
  • Use JavaScript > HTML document me "Script" add kar ke ham kisi bhi event ya Function ko kara sakte hai.

    • Q.2 What are the ways to Apply JavaScript ?

    > Webpages me JavaScript ko 3 tarike se add kar sakte hai:-

    1. Embedding code: <script>... </script> tag ke bich mein JavaScript code add karna.
    2. External Files: JavaScript ki ek alag external file bana kar usme JavaScript add karna.

    Q.3 Features of Javascript?

  • Features of Javascript:-
  • Dynamic Type - Dynamiclly ham HTML or CSS dono ko hi update or change kar sakte hai.
  • Functional Style - Isse ham kisi bhi Function ko add kara sakte hai.
  • Single-threaded language -Yeh Single-threaded language hai jo ek baar me sirf ek hi kam karti hai ek kam ko pura karne ke baad hi agle kam ko karti hai.
  • Client-Side Validation - Client-Side Validation kisi bhi form ko bharne ke liye hamse data leta hai or check karta hai ki diya gaya data sahi hai or match karta hai he ki nahi.
  • Light Weight Scripting language - Iska code ya application chal raha ho to kam storage use me leta hai.
  • Case Sensitive

    • Q.5 How many type of varibale in js?

    Three types of Variable -

    1. Var Var ko Re-declear or Re-assign dono kar sakte hai.
