Variables in Js

> Variables ka use ham kisi bhi information ko store karane ke liye karte hai.

There are 3 ways to declare a Variable

[1] Var

Var => Var ko Re-declear or Re-assign dono kar sakte hai.

[2] let

Let => Let ko Re-declear nahi kar sakte hai lekin Re-assign kar sakte hai.

[3] Const

Const => Const ko Re-declear or Re-assign dono hi nahi kar sakte hai.
