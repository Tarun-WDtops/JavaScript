> JavaScript ek event based programming languages hai, Jo ek Webpage me functionality ko add karta hai(Example => Clickable buttons,Popup menus,complex animation etc.)

> Yeh "Single Threaded Language" hai jo ek time par ek hi kam karti hai, Or yeh step by step kam karti hai, apna pahla kam pura karne ke baad hi yeh dusra kam shuru karti hai.

> JavaScript is a Dynamically typed language

> Static type or Dynamic type ye 2 type ki language hoti hai JavaScript mein => [1] Static type language :- Static type yeh hai ki ham isme jo bhi chiz karwayenge to coderun karwane ke baad me hame wesi hi milegi usme kisi bhi prakar ka koi change nahi hota hai. [2] Dynamic type language:- Dynamic type me code run karwane ke baad me hame isme change dekhne ko milte hai, isme store variables change hote hai.
