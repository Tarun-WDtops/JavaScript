Intro Of Variables

> A Variable is a container for a value, It's used to store value.

> Variables are containers for storing information. Creating a variable in JavaScript is called "Declaring" a variable: After the declaration, the variable is empty (it has no value).

> There are Two types of Variables and One constant -

1. Var -:

  • Var is an old variable declaration.
  • We can deaclare or assign Var.
  • We can also Re-declare or Re-assign.

    • Example :-

    var a;
    var a : 20; Initialization & Declaration or Assign.
    var a : 15; Re-declaration is possible.
    a = 25; Re-assign is possible.
    console.log(a);

    2. Let -:

  • Let is a modern variable declaration.
  • We can declare or assign Let.
  • We Can't Re-declare a Let, but Re-assign is possible.

    • Example :-

    let tarun_marks = 80 Initialization & Declaration or Assign let.
    tarun_marks = 40 Re-declaretion is not possible.
    tarun_marks = 70 Re-assign is possible.
    console.log(tarun_marks);

    3. const -:

  • We can deaclare or assign Const.
  • We can't Re-declare a Const, but Re-assign is possible.

    • Example :-

    const fav_Person = "tarun"; // Initialization and Declaration or Asignemnt
    // const fav_Person = "rahul" // Re-Declaration is not possible //
    fav_Person = "deepak" // Re-Assign is not possible
    console.log(fav_Person);
